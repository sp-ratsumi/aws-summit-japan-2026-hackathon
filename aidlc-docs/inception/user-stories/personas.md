# Personas — アフターファイブ

**Version**: 2.0
**Last Updated**: 2026-05-08
**Scope**: 本プロジェクトのコアターゲットユーザー 4 ペルソナ (Q2=C, Q7=C による詳細構成)

---

## Overview

「アフターファイブ」は、定時を過ぎてもダラダラ **仕事モード** に居残ってしまう働き手を **ダメモード** へと堕とすサービス。
世間的には「生産性を下げる」と呼ばれる **ダメな欲望** (家族・推し・食事・酒・趣味・ペット) こそが明日の創造性の源であるという仮説に立ち、定時 1 時間前から "ダメな未来" を視界に差し込み、定時 = 堕落ゲートで退勤動線に一気に叩き込む。

以下 4 ペルソナは、**「帰れない理由 = 仕事モードの粘着性」と「帰ったら待っているダメな欲望」** の組み合わせで設計しています。各ペルソナに対して「刺さるダメな欲望カテゴリ (D1〜D7)」を明示します。

| # | 名前 | タイプ | 帰れない理由 (仕事モード粘着) | 帰ったら待っているダメな欲望 |
|---|---|---|---|---|
| P1 | 田中 健太 | 残業 SE | タスク管理の責任感 | D5 サウナ・ジム / D4 ビール / D6 天気 |
| P2 | 佐藤 美咲 | 趣味事務職 | 「キリの良いところまで」病 | D2 推し配信 / D5 アニメ / D3 カフェ / D4 ワイン |
| P3 | 鈴木 翔 | ペット飼い営業 | 顧客対応の切れ目がない | D1 猫のミケ + 家族 / D3 夕食 / D4 晩酌ハイボール |
| P4 | 山田 リョウ | ゲーマー推し活エンジニア | スプリント末の駆け込み | D5 ゲーム / D2 推し新曲 / D4 サワー |

## P1: 田中 健太 (Tanaka Kenta) — 残業 SE

### Basic Profile
- **年齢**: 32 歳
- **性別**: 男性
- **職業**: Web 開発 SE (中堅 SIer 所属)、リード 1 歳
- **居住地**: 東京都 品川区
- **使用路線**: JR 京浜東北線・山手線
- **IT リテラシー**: 高 (エンジニア本人)
- **使用デバイス**: **メイン Windows ノート (会社支給)**、自宅は MacBook Air、iPhone
- **家族構成**: 独身、一人暮らし

### 1 日のタイムライン (仕事モード粘着パターン)
| 時刻 | 行動 |
|---|---|
| 07:30 | 起床、Twitter 開く |
| 09:30 | 出社、定時業務開始 |
| 12:00 | 社内チャットでランチ決定 |
| 13:00 | 午後タスク突入、集中してしまう (仕事モード深化) |
| 17:00 | ▶ ここで「帰る気持ち」を失う (レビュー依頼が届いたり) = 仕事モード固着 |
| 18:00 | ▶ 定時だが「あと 30 分…」が始まる (本来はダメモードに突入すべき時刻) |
| 19:30 | ようやく退勤、サウナに寄るか迷う (ダメになり切れない) |
| 21:00 | 帰宅、コンビニ飯 (中途半端なダメ) |

### Pains (仕事モードを離脱できない症状)
- レビュー依頼や PR 確認に "ちょっとだけ" 対応しているうちに定時を過ぎる
- 定時で帰るのが「サボっているように見える」自意識 → 堕ちる決断ができない
- サウナに行きたい日に限って 19 時以降になる → 翌日の回復量が足りない

### Gains (帰ったら嬉しいダメな欲望)
- **D5 サウナ**: 近所のサウナ (18:30 in が最適スケジュール)
- **D5 ジム**: 筋トレで脳内ドーパミン
- **D4 酒**: サウナ上がりのビール (キンキン)
- **D5 趣味**: ガジェット記事チェック

### Typical Overtime Pattern
- 水・木に残業が集中 (週頭に設計、水木で実装が終盤にさしかかる)
- 月次リリース前日は 21 時超え

### 堕落トリガー (Motivations & Triggers)
- 「サウナに行けるなら帰る」が最強トリガー
- **D6 天気** (雨予報) が決定打になりがち (「今帰らないと濡れる」煽り)
- **D4 酒** コンテンツで「キンキンのビール」画像が出ると 80% 帰る
- 同僚の退勤ポストが地味に効く

### 明日の創造性 (本人仮説)
- サウナ + ビール + 早寝 で翌朝の設計アイデアが冴える実体験あり
- 「ダメになった翌日の方がコードレビューの筋が見える」ことを自認

### 関連 Epic / Story (Primary ダメ欲望)
- E1 オンボーディング
- E2 ダメな未来のリマインド体験 (D5 サウナ/ジム、D4 酒、D6 天気)
- E3 堕落ゲート (定時オーバーレイの主利用者)

## Persona Design Notes

- 4 体のペルソナは **ダメな欲望カテゴリ D1〜D6 の網羅** を担保する形で設計しています (家族系、推し系、食系、酒系、趣味系、天気/通勤系)
- **仕事モード粘着の理由** がそれぞれ異なるため、堕落への誘導タイミング・文言トーンをペルソナごとに最適化できる検証材料になります
- 使用デバイスは全員 **会社支給 Windows ノート** または **私物 Mac** を含み、Tauri (Windows/macOS) + Web PWA の両対応が正当化されます
- **D4 酒カテゴリ** は全ペルソナに何らかの形で刺さる設計 (4 ペルソナそれぞれに異なる酒ジャンルを紐付け)
- **D7 睡眠・休息** は MVP 対象外。Full フェーズで全ペルソナに共通する「翌日の創造性のための最上位投資」として追加予定
- MVP フェーズでは P1 + P3 のペア (SE + ペット飼い) を最優先ターゲットとし、フル版で P2 + P4 を深掘りすることを推奨
- 各ペルソナに「明日の創造性」セクションを設け、コンセプト仮説をユーザー視点で裏付ける
