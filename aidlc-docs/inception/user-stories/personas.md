# Personas — アフターファイブ

**Version**: 1.0
**Last Updated**: 2026-05-07
**Scope**: 本プロジェクトのコアターゲットユーザー 4 ペルソナ (Q2=C, Q7=C による詳細構成)

---

## Overview

「アフターファイブ」は、定時が過ぎても帰れずダラダラ働いてしまう働き手に、定時 1 時間前から「別の未来」を差し込んで強制的に退勤させるサービス。以下 4 ペルソナは、**「帰れない理由」と「帰ったら待っている幸せ」の組み合わせ** を切り口に設計しています。

| # | 名前 | タイプ | 帰れない理由 | 帰ったら待っているもの |
|---|---|---|---|---|
| P1 | 田中 健太 | 残業 SE | タスク管理の責任感 | サウナ・健康管理 |
| P2 | 佐藤 美咲 | 趣味事務職 | 「キリの良いところまで」病 | 推しの配信・アニメ |
| P3 | 鈴木 翔 | ペット飼い営業 | 顧客対応の切れ目がない | 猫のミケ + 家族 |
| P4 | 山田 リョウ | ゲーマー推し活勢 エンジニア | スプリント末の駆け込み | ゲームセール・推しの新曲 |

## P1: 田中 健太 (Tanaka Kenta) — 残業 SE

### Basic Profile
- **年齢**: 32 歳
- **性別**: 男性
- **職業**: Web 開発 SE (中堅 SIer 所属)、リード 1 歳
- **居住地**: 東京都 品川区
- **使用路線**: JR 京浜東北線・山手線
- **IT リテラシー**: 高 (エンジニア本人)
- **使用デバイス**: **メイン Windows ノート (会社支給)**、自宅は MacBook Air、iPhone
- **家族構成**: 独身、一人暮らし

### 1 日のタイムライン
| 時刻 | 行動 |
|---|---|
| 07:30 | 起床、Twitter 開く |
| 09:30 | 出社、定時業務開始 |
| 12:00 | 社内チャットでランチ決定 |
| 13:00 | 午後タスク突入、集中してしまう |
| 17:00 | ▶ この辺で「帰る気持ち」を失う (レビュー依頼が届いたり) |
| 18:00 | ▶ 定時だが「あと 30 分…」が始まる |
| 19:30 | ようやく退勤、サウナに寄るか迷う |
| 21:00 | 帰宅、コンビニ飯 |

### Pains (帰れない / 困っていること)
- レビュー依頼や PR 確認に "ちょっとだけ" 対応しているうちに定時を過ぎる
- 定時で帰るのが「サボっているように見える」自意識
- サウナに行きたい日に限って 19 時以降になる

### Gains (帰ったら嬉しいこと)
- 近所のサウナ (18:30 in が最適スケジュール)
- ジムで筋トレ
- 趣味のガジェット記事チェック

### Typical Overtime Pattern
- 水・木に残業が集中 (週頭に設計、水木で実装が終盤にさしかかる)
- 月次リリース前日は 21 時超え

### Motivations & Triggers
- 「サウナに行けるなら帰る」が最強トリガー
- 天気 (雨予報) が決定打になりがち
- 同僚の退勤ポストが地味に効く

### 関連 Epic / Story
- E1 オンボーディング、E2 日中のリマインド体験 (サウナ / ジム / 天気カテゴリ)、E3 退勤動線

- **使用デバイス**: **会社支給 Windows デスクトップ**、自宅は iPad Pro + iPhone
- **家族構成**: 実家暮らし (両親と)、猫 1 匹 (実家の黒猫「ソラ」)

### 1 日のタイムライン
| 時刻 | 行動 |
|---|---|
| 07:00 | 起床、朝食、推しの SNS チェック |
| 08:30 | 出社 (フレックスなし、きっちり 9 時) |
| 12:00 | 同期とランチ |
| 15:00 | お茶休憩、Amazon チラ見 |
| 17:30 | ▶ そろそろ帰る気満々、でも係長の相談で止まる |
| 18:00 | ▶ 定時、キリの良いところまで着地 |
| 18:30 | 退勤、カフェに寄ってアニメ 1 話 |

### Pains
- 係長からの「ちょっと確認」で退勤が 15 分ずつ遅れる
- 推しの配信開始 (大抵 21:00) までにお風呂とご飯を終わらせたい

### Gains
- 推しの配信を生で見る (アーカイブではなく)
- アニメの最速視聴
- 近所のスタバ限定ラテ
- 月末セールのゲーム購入

### Typical Overtime Pattern
- 毎月 25 日 (給与日) 前後の勤怠集計週で忙しい
- 基本は 18:00〜18:30 の短時間残業が積み重なる

### Motivations & Triggers
- 「推しの配信まで X 時間」の可視化に弱い
- 家のソラ (猫) の写真にやられる
- 推しのセール情報 (関連グッズ)

### 関連 Epic / Story
- E1 オンボーディング、E2 日中のリマインド体験 (推し / アニメ / カフェ)、E4 家族・ペット写真管理

## Persona Design Notes

- 4 体のペルソナは **コンテンツカテゴリの網羅** を担保する形で設計しています (趣味系、家族系、仕事疲れ系、推し系)
- **帰れない理由** がそれぞれ異なるため、コンテンツ提示のトーン・タイミングをペルソナごとに最適化できる検証材料になります
- 使用デバイスは全員 **会社支給 Windows ノート** または **私物 Mac** を含み、Tauri (Windows/macOS) + Web PWA の両対応が正当化されます
- MVP フェーズでは P1 + P3 のペア (SE + ペット飼い) を最優先ターゲットとし、フル版で P2 + P4 を深掘りすることを推奨
